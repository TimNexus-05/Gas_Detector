# Methane Gas Detector System with an Alarming System

## Overview
A simple embedded system that detects gas levels using an MQ-2 sensor, displays the status on an OLED screen  and triggers a buzzer when unsafe levels are reached.

## Features
- Gas level monitoring (MQ-2)
- OLED display output (SSD1306)
- Buzzer alert for dangerous levels

## Hardware
- ESP32-C3 Mini  
- MQ-2 Gas Sensor  
- SSD1306 OLED (I2C)  
- Buzzer  

## Usage
1. Click run in you wokwi simulator
2. Monitor the output on the OLED display



## Project Structure
- `main` → stable release  
- `integration` → combined system testing  
- `hardware` → circuit design & wiring  
- `features` → active development
